# Skill Gap Navigator - Agent Skills

## Skill 1: Add Skill Input
Ability to create input fields and manage dynamic skill lists.

## Skill 2: Skill Gap Analysis
Compare user skills with predefined role requirements and identify:
- missing skills
- weak skills

## Skill 3: Roadmap Generation
Generate a 4-week learning roadmap based on gaps:
- weekly focus
- prioritized skills

## Skill 4: Local Storage Management
Save and restore:
- target role
- skills
- roadmap

## Skill 5: UI Component Generation
Create clean, reusable React components using Tailwind CSS.

## Skill 6: Export Summary
Generate a structured summary and copy to clipboard.

## Skill 7: Testing Automation
Write Playwright tests to validate:
- user input flow
- roadmap generation
